#### 项目说明

本项目是一个基于`NestJs`的模板项目。


#### 运行项目的环境准备
全局安装`nest-cli`,推荐使用`pnpm`安装。本项目`@nestjs/cli`版本为10.4.5，
并且使用`pnpm`作为包管理工具。
```bash
pnpm install -g @nestjs/cli
```

#### 项目启动
**安装依赖**
```bash
pnpm install
```

**启动项目**
```bash
pnpm run start:dev 
```

**打包项目**
```bash
pnpm run build
```

**访问项目**
```bash
http://localhost:8080
```
